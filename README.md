Attribute category from a key code or character.

# Install

```
npm i ascii-category
```

# Usage

```
import * as ascii from 'ascii-category'
```

or if you're old-school:

```
var ascii = require('ascii-category')
```

# API

```
> const key = 48
> ascii.isDigit(key)
true
```

```
> event.which === ascii.Enter
true | false
```
